# shiny_fullstack

As an international student at Sewanee, this project really hits home. This is my data story #5 for ESCI 222 course. It is a comprehensive deepdive on major choices for international students, what influences their choice, what are they planning to do post graduation and how confident are they in their majors.

There are 3 parts to this Data Story.

Part 1: Data Entry Dashboard

This is a dashboard that I made using shiny. It is extremely helpful to input data as quick as possible. Dashboard contains some key questions which are the base of the project.

- Which country are you from?
- What is your major?
- What year are you in?
- What is the most significant influence/motivation in choosing your major?
- What do you plan to do after graduation?
- On a scale of 1-5, how confident are you in your major?

Part 2: Data Visualization Dashboard

This is an Rmd dashboard which takes in the live dataset from part 1 data entry dashboard and visualizes the data in a story format. It includes 3 tabs with 4 boxes in each tab.

Tab 1 Beginning
- Tab 1 introduces the project in one box, shows how to navigate the dashboard in one box and other two boxes visualizes the major choice and derives insights from it.

Tab 2 Motivation & Plans
- Left column with two boxes show the motivation behind choosing major and the insights that can be generated from it.
- Right column with two boxes illustrates plans for life after Sewanee and illuminates on it through words.

Tab 3 Confidence & Conclusion
- Left column includes confidence plot with it's data
- Right column shows insights generated from the confidence plot and the ultimate conclusion of the data story/project.
